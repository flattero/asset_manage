Import Assets

                    
                

                

                    

                        
Upload a CSV file to import multiple assets at once. The CSV should use the same column headers as the export function.

                        

                            Supported columns: Asset ID, Device Name, Device Type, Brand/Model, Serial Number, Location, Assigned To, Department, Ownership Type, Status, Warranty End, Lease Expiry Date, Purchase Date, Purchase Cost, Operating System, MAC Address, Barcode, QR Code, Supplier, Notes


                            Tips:
                            
    
                                
  • Export existing data first to see the correct format
    • 
                                
  • Asset ID is optional - system will auto-generate if blank
    • 
                                
  • Duplicate Asset IDs will be skipped
    • 
                                
  • Required fields will use defaults if missing
